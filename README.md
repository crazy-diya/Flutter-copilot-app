# Flutter-copilot-app

A Flutter application featuring a splash screen and Hello World home screen.

## Features

- **Splash Screen**: Animated fade-in screen with app icon and loading indicator, displayed for 3 seconds on startup.
- **Hello World**: Home screen displaying "Hello, World!" after the splash screen.

## Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (>=3.0.0)

### Running the App

```bash
flutter pub get
flutter run
```

### Running Tests

```bash
flutter test
```

## Project Structure

```
lib/
  main.dart         - App entry point
  splash_screen.dart - Animated splash screen
  home_screen.dart  - Hello World home screen
test/
  widget_test.dart  - Widget tests
```
